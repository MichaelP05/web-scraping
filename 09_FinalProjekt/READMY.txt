# Automated Job Application Bot

Ця програма автоматизує процес пошуку вакансій, парсингу даних про них та надсилання заявок на вакансії через веб-сайти, наприклад, Xing. Вона також може використовувати проксі-сервери, випадкові `User-Agent` та інтегрується з базою даних SQLite.

## Функціонал
1. **Пошук вакансій на платформах**:
   - Автоматичний пошук вакансій за заданими URL-адресами.
   - Збір інформації про вакансії: назва, посилання.
   - Збереження зібраної інформації у форматах JSON та SQLite.

2. **Надсилання заявок**:
   - Автоматичне заповнення форм на веб-сайтах роботодавців.
   - Завантаження файлів резюме (CV) та супровідного листа.
   - Відправка даних через веб-форми або кнопки "Apply".

3. **Електронна пошта**:
   - Можливість відправки листів із вкладеннями для відгуків на вакансії.

4. **Інструменти для роботи з HTTP-запитами**:
   - Використання випадкових `User-Agent` для уникнення блокування.
   - Підтримка проксі-серверів.

  lxml

  requests
  json
  sqlite3

├── main.py               # Основний файл програми

├── vacancies.json        # Файл із вакансіями

Як використовувати
1. Клонування репозиторію

Копіювати код
git clone https://github.com/MichaelP05/web-scraping/09_FinalProjekt.git
cd job-application-bot


2. Установка залежностей


Копіювати код
pip install -r requirements.txt

3. Підготовка змінних середовища
Для безпечного зберігання паролів використовуйте .env файл:


Копіювати код
EMAIL_PASSWORD=your_password

4. Запуск програми
Для пошуку вакансій:
bash

Копіювати код
python main.py --search

Для надсилання заявок:
bash

Копіювати код
python main.py --apply

Конфігурація
EMAIL: ваша електронна пошта (вказана в коді або змінній середовища).
CV_PATH: шлях до вашого резюме (PDF, DOC).
Проксі: визначено у методі get_proxy.
Основні файли
main.py
Містить основну логіку:

parse_xing: Збір інформації про вакансії.
process_vacancies: Надсилання заявок на вакансії.
send_email: Функція для відправки електронних листів.
user_agents.json
Кешований список User-Agent для рандомізації HTTP-запитів.

vacancies.json
JSON-файл, де зберігаються зібрані вакансії.

vacancies.db
База даних SQLite для трекінгу оброблених вакансій.
